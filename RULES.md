# RULES — SOURCE OF TRUTH

## מה המשתמש דרש
- יום גדול בדף הראשי
- תאריך לועזי גדול
- תאריך עברי גדול
- אירועי היום והערות היום מתוך JSON אמיתי
- כפתורי ניווט עובדים באמת: יומי / שבועי / חודשי / עורך
- שבת ורוד
- פרשת שבוע אמיתית
- בלי טקסטי דמו
- כל דבר חייב להיות מסונכרן לריפו
- בדף הכללים חייב להיות תמיד: מה בוצע ומה עוד נשאר

## מה בוצע בפועל
- ריפו אחד בלבד
- GitHub Pages פעיל
- דף בית יומי אמיתי עם יום + תאריך לועזי + תאריך עברי
- טעינת JSON אמיתית
- קובץ לוגיקה משותף core.js
- קובץ עיצוב משותף ui.css
- דף שבועי מיושר לעיצוב ולניווט של המערכת
- דף חודשי מיושר לעיצוב ולניווט של המערכת
- שבת ורוד משולבת בלוגיקה
- פרשת שבוע משולבת בלוגיקה

## מה בוצע חלקית
- ניווט קיים אבל עדיין דורש אימות חזותי בטלפון
- שבת + פרשה קיימים בקוד אך עדיין דורשים אימות חזותי בכל התצוגות
- RULES מעודכן ידנית לפי המצב, עדיין לא מתעדכן אוטומטית

## מה עדיין נשאר לבצע
- אימות חזותי מלא בטלפון לכל המסכים
- ליטוש פרימיום מלא ל-UI
- חיזוק העורך לרמה גבוהה יותר
- אוטומציה מלאה של בדיקות ועדכון RULES
- ניקוי שאריות אם יתגלו במסכים נוספים
