# RULES — SOURCE OF TRUTH

## זהות המערכת
- זהו יומן פרטי אישי מבוסס GitHub.
- זה לא יומן גוגל.
- אין דמו בכלל.
- הריפו היחיד הוא `yanivmizrachiy/github-journal`.

## מה המשתמש דרש
- בדף הראשי יופיע בגדול היום של היום.
- בדף הראשי יופיע בגדול התאריך הלועזי.
- בדף הראשי יופיע בגדול התאריך העברי.
- בדף הראשי יוצגו אירועי היום במשבצות גדולות.
- יהיו כפתורי ניווט צבעוניים לתצוגה יומית, שבועית וחודשית.
- תהיה כתיבה חכמה ברמה גבוהה.
- כל יום יישמר כקובץ אמיתי תחת `entries/YYYY-MM-DD.json`.
- תמונות יישמרו כקבצים אמיתיים תחת `uploads/`.
- כל שבת תהיה צבועה בוורוד.
- בכל שבת תופיע פרשת השבוע באופן אמיתי.
- אין להשתמש בטקסטים או יכולות דמו.
- RULES.md חייב להציג תמיד: מה דרש המשתמש, מה בוצע, ומה נשאר.

## מה בוצע בפועל
- נשאר ריפו אחד בלבד: `github-journal`.
- GitHub Pages מוגדר.
- קיימת שמירה אמיתית של JSON.
- קיים דף בית.
- קיים עורך.
- קיים דף היסטוריה.
- קיים דף חודשי.
- קיימת תשתית בסיסית ל-PWA.

## מה בוצע חלקית
- דף הבית שופר, אבל עדיין לא הוכח שהוא עומד ב-100% מכל דרישות הפרימיום.
- העורך קיים, אבל עדיין דורש שיפור רמת UI וחוכמת הכתיבה.
- ניווט בין תצוגות קיים חלקית.

## מה עדיין נשאר לבצע
- לוודא שדף הבית עומד בדיוק מלא בדרישת הפרימיום.
- להשלים תצוגה שבועית אמיתית.
- לחבר ניווט מלא בין יומי, שבועי וחודשי.
- לוודא ששבת ורוד + פרשה עובדים באמת בכל התצוגות.
- לחזק את שמירת התמונות בפועל ולאמת אותה.
- לחזק את RULES כמערכת ניהול חיה ומתעדכנת.
