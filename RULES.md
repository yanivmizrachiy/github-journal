# RULES — SOURCE OF TRUTH (REAL)

## דרישות המשתמש
- דף יומי: יום + תאריך עברי + לועזי + אירועים + הערות
- ניווט: יומי / שבועי / חודשי / כתיבה
- הפרדה: HTML / CSS / JS
- שמירת נתונים אמיתית לפי תאריך
- לוח חודשי אמיתי
- שבת ורוד + פרשת שבוע
- ראש חודש צהוב
- חגים צהוב
- חופשות מוסדות לימוד ירוק

---

## מה קיים בפועל (נבדק בקוד)
- דף יומי עובד ומציג נתונים
- ניווט בין דפים עובד
- שמירת נתונים ל-GitHub עובדת
- טעינת נתונים עובדת
- הפרדה בין HTML/CSS/JS קיימת
- לוח חודשי קיים עם grid תקין
- שבת מסומנת ורוד
- פרשת שבוע מזוהה לפי title (Hebcal)
- ראש חודש מסומן
- חגים מסומנים
- חופשות נטענות מקובץ JSON

---

## מה לא מובטח עדיין (אמת)
- פרשת שבוע תלויה ב-API חיצוני (Hebcal)
- חופשות בית ספר תלויות בקובץ ידני (לא מקור רשמי אוטומטי)
- תצוגה שבועית לא באותה רמת דיוק כמו החודשי
- לא בוצע אימות חזותי מלא בכל המסכים
